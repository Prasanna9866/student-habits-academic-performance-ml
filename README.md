# Student Habits vs Academic Performance Using Machine Learning

## Project Description
This project predicts student academic performance based on habits such as study hours, attendance, and sleep using machine learning.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## Dataset
Student habits dataset including study hours, attendance, sleep hours, and scores.

## Outcome
Predicts student academic performance using machine learning models.
